# WorldPredict — Visual System & Creative Direction

## Product Identity

WorldPredict should feel:
- competitive,
- modern,
- fast,
- social,
- premium,
- and sports-oriented.

The visual direction must combine:
- sports apps,
- esports dashboards,
- social gaming,
- modern SaaS UI.

The interface should NOT resemble:
- casino apps,
- betting platforms,
- outdated sports websites.

---

# Onboarding Experience

## Critical Requirement
The onboarding flow should create emotional engagement immediately.

---

# Suggested Flow

1. Choose country
2. Choose favorite team
3. Join or create group
4. Make first prediction
5. See live ranking

The onboarding should take less than 60 seconds.

---

# UX Philosophy

The product must optimize:
- fast interaction,
- dopamine feedback,
- competitive feeling,
- social retention,
- low friction.

Every important action should feel rewarding.
